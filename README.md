# Dr. Darnell's Fetal Strip Practice – Professional Edition

## Files
- `index.html` — main application page
- `styles.css` — visual design and print certificate styling
- `cases.js` — fetal-monitoring case bank
- `app.js` — game logic, narration, scoring, simulator, reports, and certificate

## Preview on your computer
1. Download and unzip this package.
2. Open the extracted folder.
3. Double-click `index.html`.
4. Use Microsoft Edge or Google Chrome.

Because all case data is stored in `cases.js`, the app works locally without a web server.

## Publish with GitHub Pages
1. Create a public GitHub repository.
2. Upload all four files to the repository root.
3. Open **Settings → Pages**.
4. Choose **Deploy from a branch**.
5. Select the `main` branch and `/root`.
6. Save.
7. Use the GitHub Pages address in Blackboard.

## Updating the app later
Keep the same GitHub repository and replace only the files that changed. The Blackboard link will remain the same.

## Educational note
This application is an educational simulation. Clinical practice should follow current institutional policies, provider orders, and emergency protocols.
